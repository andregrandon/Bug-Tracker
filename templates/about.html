About Bug Tracking System

The bug tracking system (also known as: ITS, trouble bug system, support bug, request managment or incident bug system) is a computer software package that manages and maintains lists of bugs, as needed by an organization, bug tracking systems are commonly used in an organizations customer support call center to create, update, and resolve reported customer bugs, or even bugs reported by that organzations other employees. A support bug should include vital information for the account involed and the bug encountered. This system also contains knowledge base containing information on each customer, resoultuions to common problems and other such data. Consisistent use of an bug tracking system is considered one of the "hallmarks of a good software team"
